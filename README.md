# Smart Question Maker

Smart Question Maker is an intelligent, multi-tenant educational platform 
designed to automate the process of question preparation, examination, and 
evaluation for coaching centers and academic institutions.

## Overview
Teachers spend a lot of time manually preparing MCQ questions. This system 
uses AI (NLP) to automate MCQ generation from uploaded study materials such 
as slides, notes, and documents.

## Key Features
- 🤖 AI-powered MCQ generation in 4 modes (Manual, Random, Guided AI, Zero-Shot)
- 🔐 Role-based authentication (Super Admin, Coaching Admin, Teacher, Student, Parent)
- 📝 Question bank management with difficulty levels (Easy, Medium, Hard)
- 📄 Automated question paper export in PDF format (Set A/B)
- 📊 Performance analytics and result management
- 🔔 Automated notifications via SMS and Email
- 💳 Subscription and payment management

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **AI Engine:** OpenAI API, LangChain
- **Auth:** JWT
